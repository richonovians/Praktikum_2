Jika Anda mengklik tombol "Kirim", data formulir akan dikirim ke halaman bernama "/action_page.php".

  1. Pada method get saat kita melakukan submit, data yang dikirim akan terlihat di query parameter, sedangkan method post data yang dikirim tidak akan terlihat di query parameter, namun kita bisa lihat data tersebut dikirim melalui dev tools.
  2. Atribut target _blank untuk membuka tab baru, _self untuk membuka di halaman itu sendiri, _parent untuk membuka dalam bingkai induk, _top untuk membuka tautan dalam seluruh area jendela.
