Uji coba 2




Jadi kesimpulannya, jika kita mengisi atribut name dengan nilai sama, yang terjadi adalah kita hanya bisa memilih 1 dari 3 pilihan, karena dari 3 input tersebut memiliki 1 name yang sama. Dan jika kita mengisi atribut name dengan nilai yang berbeda, yang terjadi adalah kita bisa memilih semua, karena setiap input mempunyai name masing-masing.

Jadi kesimpulannya, dari 3 input checkbox tersebut ketika kita mengisi name dengan nilai sama, maka hasilnya tetap bisa memilih lebih dari satu pilihan atau tidak memilih sama sekali, karena input tersebut bertype checkbox. Dan juga pada label for yang di tuju bukan atribut name, melainkan pada atribut id.

Uji coba 4




  1. Pada input type color berfungsi untuk menginput warna dengan hexa, RGB dan HSL
  2. Pada input type date berfungsi untuk menginput tanggal, bulan dan tahun.
  3. Pada input type button adalah untuk menambah tombol yang berfungsi untuk alert/warning.

Uji coba 5




  1. Pada input type datetime-local berfungsi untuk menginput tanggal, bulan, tahun serta jam dengan format yyyy-mm-ddThh:mm.
  2. Pada input type month berfungsi untuk menginput bulan dan tahun dengan format yyyy-mm.
  3. Pada input type email adalah input text, namun informasi yang dimasukkan haruslah dalam format email. Jika format email tidak benar, secara otomatis form tidak akan bisa di submit.

Uji coba 6




  1. Pada input data type image bukan digunakan untuk input berupa berkas gambar, melainkan digunakan untuk membuat tombol submit namun berupa gambar.
  2. Pada input type file digunakan untuk mengupload berkas file apapun, seperti pdf, jpg, png, dll.
  3. Input type hidden artinya input datanya tidak terlihat oleh pengguna web. Biasanya input data hidden memang tidak dimaksudkan untuk diinputkan oleh pengguna, biasanya pada kasus-kasus tertentu input data hidden digunakan untuk informasi yang dibuat secara otomatis oleh web kita, dan tidak butuh diubah dan dilihat oleh pengguna.

Uji coba 7



  1. Pada input type number berguna untuk input number atau angka.
  2. Input data type range adalah input untuk number/angka namun ditentukan jangkauan minimal dan minimalnya. Ini mirip seperti input number, yang membedakannya adalah bentuk komponen inputnya tidak seperti input text, melainkan dalam bentuk slider, yang bisa di slide ke kiri untuk menurunkan nilai dan ke kanan untuk menaikkan nilai. Secara default, nilai range naik bertahap +1 atau berkurang -1. Jika kita ingin mengubah, kita bisa tambahkan atribut step.
  3. Atribut min dan max untuk mengatur berapa nilai minimal dan nilai maximal data yang diinputkan.

Uji coba 8




  1. Input type URL digunakan untuk menerima input data dalam bentuk URL, seperti alamat website pribadi, website social media, dan lain-lain. Jika format URL tidak valid, maka HTML tidak akan menerima input data URL tersebut.
  2. Input type search digunakan untuk memungkinkan pengguna memasukkan teks pendek yang akan digunakan sebagai parameter dalam operasi pencarian.
  3. Input type tel bisa digunakan untuk input informasi telepon. Namun berbeda dengan input email atau number, aturan di input tel harus kita tentukan sendiri, hal ini karena format telepon di tiap negara berbeda-beda. Kita bisa mengubah aturan format nomor telepon menggunakan atribut pattern, dimana isinya harus berisi Regular Expressions.

Uji coba 9



  1. Input type time digunakan untuk informasi jam dan menit dengan format hh:mm. Untuk membatasi minimal dan maksimal waktu yang dipilih, kita bisa gunakan atribut min dan max.
  2. Input type week digunakan untuk menerima input minggu dalam satu tahun. Total minggu dalam satu tahun berkisar antara 52-53, artinya itu adalah maksimal nilai untuk week. Format penulisan week bisa menggunakan pola yyyy-Www. Untuk membatasi minimal dan maksimal kita juga bisa gunakan atribut min dan max.
